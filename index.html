Projects

Work Project: BCU Online Paperwork for Financial System Registration

Client: Central Bank of Uruguay (BCU)

In this project, I was responsible for developing a new information entry component (shareholders tree) using React and TypeScript, focusing on client-side validations and user interactions. My role also included integrating the information with both an administration and registration system and an electronic file system that manages documents and actions signed electronically. I ensured seamless data flow and compliance with BCU standards across these systems.

On the backend, I developed RESTful services, SOA services, and POJOs, facilitating interactions between the database and frontend. I conducted comprehensive testing with SoapUI and Postman, ensuring high reliability and functionality across modules.

I managed deployment on test servers using JBoss and collaborated with a team of 3 developers and 1 analyst, using SVN for version control. Additionally, I attended weekly meetings with BCU stakeholders, providing updates, gathering feedback, and aligning on project milestones.

Work Project: AIN SIAIGU Internal Audit Management System

Client: Internal Audit of the Nation (AIN)

In this project, I was responsible for implementing various business processes within the system and customizing React components to align with the clients specific requirements. Additionally, I designed Jasper reports to display the information captured in these components. I also developed interactive dashboard panels to provide users with features based on their roles.

On the backend, I created POJOs for the interactions between the database, frontend components, and Jasper reports. I attended several targeted client meetings to address specific questions and requirements. As part of a team of 2 developers, 1 analyst, and 1 tester, we collaborated closely, using Git for version control.

Nostra Pizza Mobile and Web App

I am currently developing a mobile and web application for a pizza delivery company, Nostra Pizza. The app allows customers to place orders, track deliveries in real time, and receive notifications on the status of their orders. It also includes an admin panel that allows staff to manage orders, customer information, and inventory.

This project is being built using React Native with Expo to ensure a consistent experience across both mobile and web platforms. For the backend, I am utilizing Node.js, while the database is powered by PostgreSQL with Supabase for real-time data updates and simplified database management. I am using Android Studio to test the app on an Android emulator. I am working independently on all aspects of this project and managing code versions with Git.
